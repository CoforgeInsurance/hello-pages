# hello-pages

A static GitHub Pages site featuring AI-related content with a classic vintage theme.

## Live Site

Visit the live site at: https://coforgeinsurance.github.io/hello-pages/

## Content

This single-page website features:
- **3 AI Quotes** - Wisdom from thought leaders in artificial intelligence
- **3 AI Jokes** - Humorous takes on AI and machine learning
- **3 AI Fun Facts** - Fascinating historical and technical facts about AI
- **3 AI Prompts** - Creative prompts to try with your favorite AI assistant

## Theme

The site features a classic vintage design with:
- Serif typography (Georgia, Times New Roman)
- Warm, paper-like color palette
- Classic ornamental elements
- Responsive design for all devices

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions whenever changes are pushed to the `main` branch.
